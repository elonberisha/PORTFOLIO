# PORTFOLIO

Personal portfolio for Elon Berisha, built with Next.js, Sanity Studio, and a CMS-driven content system.

## Development

```bash
npm install
npm run dev
```

Open:

- Site: http://localhost:3000
- Studio: http://localhost:3000/studio

## Environment

Copy `.env.local.example` to `.env.local` and fill in the required values.
